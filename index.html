Como Ativar a Conexão Real

O seu dashboard já possui o link direto para a planilha Head <> Cabeça de Lab. Para que o seu site no GitHub Pages consiga ler os dados, você deve permitir o acesso externo no Google Drive:

  1. Abra a sua planilha no Google Drive.
  2. Clique no botão azul Compartilhar no canto superior direito.
  3. Em *Acesso Geral*, mude de *Restrito* para "Qualquer pessoa com o link" como Leitor.
  4. Salve as alterações.

Acesso Seguro:

Esse método só permite que o site "leia" os dados consolidados, sem qualquer autorização de alteração, exclusão ou acesso a outros arquivos da sua conta Google.
